Predicting Flower Prices at OCS (Ontario Cannabis Store)

I used Selenium in Python to scrape the OCS website to extract the product info for over 300 cannabis strains. After creating graphs and calculating statistics to identify the relationships among the variables (e.g. price, grams, THC/CBD amounts), I implemented Linear, Lasso, and Random Forest (RF) Regressions to estimate flower prices.

Python packages used: Pandas, Numpy, Scikit-learn, Matplotlib, Seaborn, Selenium

NYCgov Poverty Measure Data (2014) Dashboard

The NYCgov Poverty Measure Data (2014) is a census dataset published by the Mayor's Office for Economic Opportunity in NYC. I used Tableau Desktop to create three dashboards: Dashboard 1 and Dashboard 2 look at average income, whereas the Medical Spending Dashboard looks at average medical spending across different demographics based on select variables (e.g. sex, age, race, language proficiency, educational attainment, citizenship status).
